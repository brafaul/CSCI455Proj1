Date: Wed, 20 Nov 1996 19:14:57 GMT Server: Apache/1.0.3 Content-type: text/html Content-length: 1463 Last-modified: Mon, 04 Dec 1995 18:59:05 GMT Syntactic Abstraction Syntactic Abstraction Description: Traditional macros used to extend programming languages suffer from semantic difficulties due to unintended variable bindings that occur as the result of the code transformations specified by macro transformers. We have developed the concept of hygienic (lexically scoped) macros and are continuing to refine the concept of syntactic abstraction and render it usable for a wider variety of applications.
