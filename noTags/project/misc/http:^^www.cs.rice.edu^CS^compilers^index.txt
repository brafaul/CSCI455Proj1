Rice Compiler Group


Members of the Rice Compiler Group focus on compiling imperative programs for modern computer architectures. Projects in the Compiler Group include (1) the Massively Scalar Compiler Project (MSCP), which concentrates on compilers for advanced microprocessors, and (2) the Fortran Parallel Programming Systems, or Fortran Tools, project, which concentrates on compilers and tools to support machine-independent parallel programming in Fortran.
