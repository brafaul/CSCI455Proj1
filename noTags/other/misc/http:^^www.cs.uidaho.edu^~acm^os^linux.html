Date: Tue, 14 Jan 1997 19:57:27 GMT Server: NCSA/1.4.1 Content-type: text/html Last-modified: Sat, 28 Sep 1996 22:26:09 GMT Content-length: 873 Linux Linux Description Linux is an extremely popular OS for i386 machines, with ports for m68k Macintosh and Amiga machines, MIPS based machines, and DEC Alphas. One of the reasons that Linux is so popular is that its development is completely open -- anyone can contribute. Availability There are several linux distributions out there, both available on CDROM and by FTP. Linux is free, but their is a price for the distributions. This is to pay for the CDROM burning, as well as tech support. Links for more information SSC's Linux page with links to just about everywhere. Comments?
