The SimpleScalar Architectural Research Tool Set comprises compiler, assembler, linker and simulation tools for the SimpleScalar architecture. The tool set is partly derived from the GNU software development tools. It provides researchers with an easily extensible, portable, high-performance test bed for systems design. Here are slides from the simulation workshop talk I (Todd Austin) gave at the last ISCA in Santa Margherita Ligure, Italy. The talk detailed the tool set and insights we've gained from its use. How do I get the SimpleScalar Architectural Research Tool Set? The source release for version 1.0.0 should be available here in early 1996. Please send me e-mail (at austin@cs.wisc.edu) for further details. Who made the SimpleScalar Architectural Research Tool Set? Much of the compiler tools are simply ports of the GNU software development tools to the SimpleScalar architecture. The ports as well as the simulators were written by Todd Austin of the Wisconsin Multiscalar Group at the University of Wisconsin - Madison. The development of this code was supported by grants from the National Science Foundation (grant CCR-9303030 plus software capitalization supplement) and the Office of Naval Research (grant N00014-93-1-0465).
