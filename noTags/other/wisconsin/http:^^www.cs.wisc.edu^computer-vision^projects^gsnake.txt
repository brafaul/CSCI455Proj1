gsnake.tar : 3 MB. Download tar file containing GSNAKE API after you have registered.
It contains the following directories:

  src  : contains GSNAKE API class library written in C++
  example  : workable examples coded by GSNAKE API
  util  : utilities provided with GSNAKE API
  docu  : a complete set of manual entries for GSNAKE API
  lib   : contains GSNAKE API library after compilation
  include  : contains header files for GSNAKE API
  bin  : contains test images and binary source after compilation
  data  : contains some test data, mostly in sun raster format
  thesis   : contains the thesis by K.F. Lai which leads to this API
