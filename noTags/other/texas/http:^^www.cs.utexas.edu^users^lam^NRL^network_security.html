MIME-Version: 1.0 Server: CERN/3.0 Date: Tuesday, 07-Jan-97 15:55:05 GMT Content-Type: text/html Content-Length: 4987 Last-Modified: Friday, 01-Mar-96 21:21:33 GMT Network Security Network Security To enhance the security of networked systems, we proposed a security architecture together with novel protocols for authentication, called Texas Authentication Protocols (TAPs), and a new language for authorization, called Generalized Access Control List (GACL). A proof methodology for verifying authentication protocols based upon state transition semantics has been developed. A high level abstraction for secure network programming (SNP), designed to resemble a socket interface, has been implemented.
